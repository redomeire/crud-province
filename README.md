# CRUD Provinsi 📝  


## Get Started 🚀 
Website to manage province data

## Installation 🧑‍💻
1. please clone this repo `git clone https://github.com/redomeire/crud-province.git`
2. run `npm install`
3. run `npm start`
4. open localhost:3000 in the browser

## Technologies 🕹
- React Js
- Tailwind css
- Reqres api for handling authentication
 
## Features 💎
- Login 
- Register 
- Add province
- Edit province
- delete province
- search province
